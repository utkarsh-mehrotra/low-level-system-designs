# SDE3 System Designs

A consolidated repository containing multiple distributed FAANG-level Microservice architectures, refactored from legacy Object-Oriented monorepos.

## Projects Included
1. **Airline Management System** (Pessimistic Locking, Event-Driven Sagas)
2. **Car Rental System** (Date-Overlap Database Locks, Redis Global Caching)
3. **Parking Lot System** (Redis Min-Heap O(1) Spot Assignment, Async Billing)
